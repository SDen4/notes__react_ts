## Notes App (SPA)

### by Denis Skryabin

Watch on [github pages](https://sden4.github.io/notes__react_ts/)

### Stack:

```sh
html
scss
typescript
react
redux
```

#### Установить проект

```sh
$ git clone https://github.com/SDen4/notes__react_ts.git
$ cd notes__react_ts
$ yarn
```

#### Запустить dev сервер

```sh
$ yarn start
```
